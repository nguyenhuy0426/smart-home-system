# Camera Node Scaffold

This folder keeps the existing PlatformIO ESP-IDF project structure. The added stubs define where identity, descriptor exchange, RTSP stream status, and gateway-owned snapshot policy responsibilities will live after the design phase.

No camera driver logic, RTSP server implementation, BLE Mesh internals, Wi-Fi credential handling, or production OTA code is implemented here.
