# Access Control Node Scaffold

This folder keeps the existing PlatformIO ESP-IDF project structure. The added stubs define where identity, descriptor exchange, access-event normalization, credential privacy, and relay-result responsibilities will live after the design phase.

No UART fingerprint matching code, RFID driver logic, relay GPIO behavior, credential storage implementation, or production door-control logic is implemented here.
