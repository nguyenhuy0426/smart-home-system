# Environment Sensor Node Scaffold

This folder keeps the existing PlatformIO ESP-IDF project structure. The added stubs define where identity, descriptor exchange, sensor acquisition, and DHT22/CJMCU680 fusion responsibilities will live after the design phase.

No hardware driver logic, pin choices, BLE Mesh internals, or production sampling code is implemented here.
